# Sandaru Suranjaya - Personal Website

A modern, one-page personal portfolio website built with Next.js, designed to showcase my work, story, and photography.

## Features

- **Hero Section**: Interactive particle background with a glassy aesthetic.
- **Story & Purpose**: A section dedicated to my professional journey and values.
- **Memory Reel**: A curated photo gallery showcasing my photography.
- **Recommendations**: Testimonials and endorsements.
- **Behind The Lens**: Insights into my creative process.
- **Contact**: A simple way to get in touch.
- **Theme Support**: Fully responsive with Dark and Light mode support.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: CSS Modules with a custom design system
- **Fonts**: [Geist](https://vercel.com/font) (Sans & Mono)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
